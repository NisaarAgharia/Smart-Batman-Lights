# Smart-Batman-Lights
3D Printed IoT Smart Batman Lights with multiple modes and moods

## Features

- 100+ modes and mood lighting
- Voice Controlled with Alexa and Google
- Mobile App Controlled  
- Can be Controlled from anywhere in the world  
- Reacts to Music
